# フェーズシフタ

## 概要

フェーズシフタはシンセサイザーのモジュールというより、どちらかというとエフェクタとしてよく知られています。もともとは回転スピーカの効果に似た音を得るために考案されたようですが、実のところあまり似ていません。それでも、音がうねるような独特の面白い効果が得られることから重宝されているエフェクタです。回路図は次ページの通りです。
