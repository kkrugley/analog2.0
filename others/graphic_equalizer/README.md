# グラッフィック・イコライザ

## 概要

グラフィックイコライザは、多数並んだ周波数バンドごとに共振フィルタを持っており、周波数バンドごとに周波数特性を強調したり弱めたりできます。このグラフィックイコライザには、1オクターブごとの7バンドがあります。
